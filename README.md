# React Todo App

A modern Todo application built with React and Bootstrap.

## Live Demo

https://react-todo-app-theta-ebon.vercel.app

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Filtering
- Local Storage Support
- Responsive Bootstrap UI

## Technologies Used

- React.js
- JavaScript
- Bootstrap 5
- Vite
- Local Storage

## Installation

npm install
npm run dev
